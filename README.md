# Lab3
Лабораторная работа №3 Работа с Git
Цель работы: освоение системы контроля версий Git.
Выполненные задачи:
-Добавление строк в файлы и осуществление коммита.
-Сохранение разницы между коммитами в отдельный файл.
-Осуществление отката к какому-либо коммиту.
-Сливание веток и решение конфликтов
-Сохранение истории всех коммитов в отдельный файл.
Итог: Я стал разбираться в системе контроля версий Git. Получил опыт работы с ветками, решения конфликтов, осуществления коммитов и мерджей.
